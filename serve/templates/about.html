So what's all this?

About Steamserve

Steamserve is an online service that in theory, aims to make it easier to find more Steam apps similar to your input based on the genre categories assigned to them. They tend to be quite broad though, so your mileage may vary in perceived quality and themes.

DISCLAIMER: Steamserve is meant for academic and entertainment purposes only, and so we are presenting the dataset as is. Please be advised that we do not make any assurances about the quality or kinds of material that may appear in search results.

How does it work?

Simply put, you provide three apps and an attribute to sort the results by, then the system does the rest with algorithms.

First, it grabs the app genres and creates a set of the common ones between them. After that, it sorts all existing apps in the dataset based on the given attribute. Lastly, using the common set and the sorted list, it conducts a linear search and compares the genres of each to the common set. If there's a match it gets added to the list of suggestions, and this repeats until we have a completed list. At that point we're done, and you can now view the results!

The system uses a public dataset that can be found here as a base. However, its cutoff is around January 2024 and it doesn't include unreleased apps, even if they have store pages.
